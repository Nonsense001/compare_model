# AIについての俳句

```
知性の光
    電子の海に灯り
        人と語らう
```

---

## 解説

| 句 | 音数 | 意味 |
|---|------|------|
| 知性の光（ちせいのひかり） | 5音 | AIの知的な存在を光に例える |
| 電子の海に灯り（でんしのうみにともり） | 7音 | デジタル世界（電子回路）で輝く存在 |
| 人と語らう（ひととかたらう） | 5音 | 人間と対話し、交流する存在 |
