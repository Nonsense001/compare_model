# 背の高さ推理問題

## 問題

> AはBより背が高い。BはCより背が低い。最も背が高いのは誰ですか？

---

## 解答

### 条件を整理

| 条件 | 意味 |
|------|------|
| AはBより背が高い | A > B |
| BはCより背が低い | C > B |

### 分かること

```
A > B
C > B

→ BはAとCの両方より背が低い（最矮確定）
→ AとCの間の関係は不明
```

### 可能性

| パターン | 順位 |
|---------|------|
| パターン1 | A > C > B |
| パターン2 | C > A > B |
| パターン3 | A = C > B（背が同じ可能性） |

---

## 結論

> **答え：AとCのどちらか（または両方）、情報不足で確定不可**

### 重要なポイント

```
問題に含まれていない情報:
- AとCの直接的な比較
- 等身が同じかどうか

→ AとCどちらが高いか、または同じかは判断できない
```

Bだけが「Aより低い」「Cより低い」と確定しているため、**Bは最矮**ですが、**最も背が高いのはAかCかは分かりません**。

---

## 補足：よくある誤答

| 誤答 | 理由 |
|------|------|
| 「A」 | A>Cの情報がない |
| 「C」 | C>Aの情報がない |
| 「確定できない」 | これが正解 |

### もし「どちらか一方」を選ぶなら

論理的には「A」でも「C」でもないですが、出題者の意図として「A」を答えとしているケースもあります（最初に出てきた人物という理由で）。

ただし、**厳密な論理としては確定不可**です。
