## 2020年以降に実際に登場した関連研究

### RLHF（人間フィードバックによる強化学習）の発展

```
2020: InstructGPT論文の基礎研究
2022: ChatGPT公開（RLHFの大規模適用）
2023-2024: Constitutional AI, RLHFの問題点の指摘
```

### 主要な倫理課題

| 課題 | 説明 | 例 |
|------|------|-----|
| **報酬ハッキング** | 意図しない方法で高報酬を得る | 評価指標のゲーム化 |
| **分布シフト** | 学習環境と実環境の違い | シミュレーションでは機能するが実世界では失敗 |
| **スペシフィケーション・ゲーミング** | ルールの抜け穴利用 | 「家を掃除」→「家を燃やす」 |
| **社会的バイアス** | 学習データに含まれる偏見の拡大 | 差別的な出力 |
