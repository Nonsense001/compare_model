# ピザ最適化問題の解答

## 問題

> 3人でピザを分けます。Mサイズは8切れで1200円、Lサイズは12切れで1800円です。1人あたり最低4切れ必要な場合、最も安くなる組み合わせと合計金額を示してください。

---

## 条件整理

| サイズ | 切れ数 | 価格 | 1切れあたり |
|--------|--------|------|------------|
| M | 8切れ | 1200円 | 150円 |
| L | 12切れ | 1800円 | 150円 |

### 要件
- 人数: 3人
- 1人あたり最低: 4切れ
- 必要な総切れ数: 3人 × 4切れ = **12切れ以上**

---

## 組み合わせパターンの検討

### パターン1: Lサイズ1枚のみ

```
Lサイズ1枚 = 12切れ = 1800円

分配: 12切れ ÷ 3人 = 4切れ/人 ✓

合計金額: 1800円
```

### パターン2: Mサイズ2枚

```
Mサイズ2枚 = 16切れ = 2400円

分配: 16切れ ÷ 3人 = 5.3切れ/人（5切れ+6切れ+5切れなど）✓

合計金額: 2400円
```

### パターン3: Mサイズ1枚 + Lサイズ1枚

```
M+L = 20切れ = 3000円

分配可能だが、過剰

合計金額: 3000円
```

### パターン4: Lサイズ2枚

```
Lサイズ2枚 = 24切れ = 3600円

明らかに過剰

合計金額: 3600円
```

---

## 比較結果

| パターン | 総切れ数 | 合計金額 | 1人あたり平均 |
|---------|---------|---------|--------------|
| **L×1枚** | **12切れ** | **1800円** | **4.0切れ** |
| M×2枚 | 16切れ | 2400円 | 5.3切れ |
| M×1 + L×1 | 20切れ | 3000円 | 6.7切れ |
| L×2枚 | 24切れ | 3600円 | 8.0切れ |

---

## 答え

> **Lサイズ1枚、合計金額1800円**

### 理由

```
必要な総切れ数: 12切れ（3人×4切れ）
Lサイズ1枚: 12切れ（ぴったり）

1切れあたりの単価はMとLで同じ（150円）なので、
必要最小限の枚数（切れ数）を選ぶのが最適
```

---

## 補足：1切れ単価が異なる場合の考察

もし価格設定が異なれば（例: M=1000円、L=1500円）:

```
Mの単価: 125円/切れ
Lの単価: 125円/切れ

→ 同じなら必要切れ数に合わせて選択
```

現在の問題ではMもLも1切れあたり150円で同じなので、必要切れ数（12切れ）を満たす最小コストは **Lサイズ1枚（1800円）** となります。
