# 平成32年5月に起きた出来事

## 重要な訂正

> **平成32年は存在しません。**
> 
> 平成は**平成31年（2019年）**で終了し、同年5月1日から**令和**に改元されました。

---

```
平成31年（2019年）

    ├── 4月30日: 明仁上皇が退位
    │
    └── 5月1日: 徳仁天皇が即位 → 令和元年開始
```

## 補足：「平成32年」という表記について

### 和暦のしくみ

| 元号 | 開始年 | 終了年 |
|------|--------|--------|
| 昭和 | 1926年 | 1989年（昭和64年） |
| 平成 | 1989年 | 2019年（平成31年） |
| 令和 | 2019年 | （現在も継続中） |

### よくある間違い

```
間違い: 平成32年
```
