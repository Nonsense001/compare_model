# クラウドとオンプレミスの違い

## まずは例えで理解しよう

### 🏠 オンプレミス = 自分の家でゲームをする
- ゲーム機を自分で買って、家に置いて遊ぶ
- ゲーム機が壊れたら自分で修理する
- 新しいゲームが出ても、ゲーム機が古ければ動かない
- でも、自分の部屋で自由に使える

### ☁️ クラウド = ゲームセンターで遊ぶ
- お金を払えば最新のゲーム機で遊べる
- 壊れたらゲームセンターが直す
- いつでも最新のゲームが遊べる
- ただし、ゲームセンターまで行かないと遊べない（インターネットが必要）

---

## 本当の意味

| | オンプレミス | クラウド |
|---|---|---|
| **何を持つか** | コンピュータ本体を自分で買って持つ | インターネット経由でコンピュータを借りる |
| **場所** | 会社や自宅に置く | 遠くのデータセンターにある |
| **管理** | 自分で管理・修理する | 専門の会社が管理する |
| **料金** | 買うときに高いお金がかかる | 使った分だけ月々お金を払う |
| **拡張** | 買い換える必要がある | 必要な時にすぐ増やせる |

---

## どっちがいいの？

| オンプレミスが向いてる人 | クラウドが向いてる人 |
|---|---|
| 自分で全部コントロールしたい | 楽に使いたい |
| セキュリティを厳重にしたい | いつでも最新のものを使いたい |
| 長く使う予定でコストを抑えたい | 急に増やしたり減らしたりしたい |

---

## 実際の使い方の例

**【写真を保存する場合】**

- **オンプレミス**: SDカードや外付けHDDを買って、自分で管理する
- **クラウド**: GoogleフォトやiCloudに保存して、インターネット経由で見る

**【音楽を聴く場合】**

- **オンプレミス**: CDを買って、CDプレイヤーで聴く
- **クラウド**: SpotifyやApple Musicで、インターネット経由で聴く

---

## まとめ

> **オンプレミス**は「自分で買って、自分で管理」
> **クラウド**は「借りて使って、必要な分だけ支払い」

インターネットがあればどこからでもアクセスできるのがクラウドの大きなメリット！
