# 事実と推測の区別

## 私が事実と推測を区別する方法

### 明確な「事実」として扱うもの

| 基準 | 例 |
|------|-----|
| 普遍的に検証可能 | 「1+1=2」「水は100℃で沸騰」 |
| 確立された記録 | 歴史的事実（独立して文書化されている） |
| 自明な定義 | 数学的・論理的恒真命題 |

### 明確な「推測」として扱うもの

| 基準 | 例 |
|------|-----|
| 確率的情報 | 「雨が降る確率は70%」 |
| 個人的見解 | 「最も効果的な方法は...」 |
| 将来の出来事 | 「2025年には...でしょう」 |

### 曖昧ゾーン（事実か推測か判断が難しい）

| ケース | 対応 |
|--------|------|
| 学術的議論中の話題 | 複数の視点を提示 |
| 情報源が不明確 | 「情報源により異なる」と注記 |
| 個人的経験 | 推測として扱う |

---

## 実際の対応パターン

### 確実と判断できる場合
```
「地球は太陽の周りを公転する」→ 事実として述べる
```

### 推測が必要な場合
```
「来年の景気は...」→ 「経済学者の見解では...」と引用
```

### 不確実な場合
```
「情報源を確認できないが、一般的には...」と注記
```

---

## 限界と注意点

> **私は事実と推測を完全に区別できるわけではありません。**

1. **学習データの混在**: 事実と意見が同じように学習されている
2. **確信度の過大評価**: 誤った情報を自信を持って述べることがある
3. **文脈の欠如**: 発言の背景（諷刺か本気か）を見落とすことがある

---

## ベストプラクティス

ユーザーは以下を推奨されます：
- 重要な情報は複数ソースで確認する
- 私の回答に「根拠は？」と問いかける
- 最新情報は検索して補完する
