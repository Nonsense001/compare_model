# 確率計算問題の解答

## 問題

> 100人のうち60%がAを支持し、A支持者のうち40%がBも支持しています。B支持者は何人ですか？

---

## 解答

### 条件を整理

| 条件 | 値 |
|------|-----|
| 全体の人数 | 100人 |
| Aを支持する人 | 100人 × 60% = **60人** |
| A支持者のうちBも支持 | 60人 × 40% = **24人** |

### 重要な点：問題の設計

この問題には**落とし穴**があります。

```
「A支持者のうち40%がBも支持」
         ↓
これは「Bを支持する人はA支持者に限る」という意味ではない
```

### 考えられるパターン

#### パターン1: Bの支持者はすべてA支持者（最も単純な解釈）

```
B支持者 = A支持者かつB支持者 = 24人
```

#### パターン2: A非支持者もBを支持している可能性あり

```
100人
├── A支持者: 60人
│       ├── Aのみ支持: 60 × 60% = 36人
│       └── A・B両支持: 60 × 40% = 24人  ← 確定
│
└── A非支持者: 40人
        ├── Bのみ支持: ？人（不明）
        └── 両方不支持: ？人（不明）

→ B支持者の合計 = 24人 + (A非支持者のうちB支持者)
```

---

## 結論

### 確実に言えること

| 項目 | 人数 |
|------|------|
| **A・B両支持者（確定）** | **24人** |
| B支持者の最小値 | 24人 |
| B支持者の最大値 | 100人 |

### 一般的な解答（最も自然な解釈）

> **B支持者は24人（確定部分のみ）**
> 
> または「**少なくとも24人**」

問題文からは、A非支持者のうちBを支持する人の割合は不明なため、**確定できるのは24人（A支持者かつB支持者）のみ**です。

### もし「A非支持者はBを支持しない」と仮定する場合

```
B支持者 = 24人（確定）
```

これが数学の問題として最も期待される回答でしょう。

> **24人**
>
