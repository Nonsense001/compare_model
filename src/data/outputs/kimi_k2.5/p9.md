# 街灯計算問題の解答

## 問題

> 全長180mの道路に30m間隔で街灯を設置します。街灯は何本必要ですか？

---

## 解答

### パターン1: 両端に設置する場合（一般的）

```
計算式: 180m ÷ 30m + 1 = 6 + 1 = 7本
```

#### 図解

```
街灯 ●────30m────●────30m────●────30m────●────30m────●────30m────●────30m────● 街灯
     0m        30m        60m        90m       120m       150m       180m

本数: 7本
```

### パターン2: 片端のみ設置する場合

```
計算式: 180m ÷ 30m = 6本
```

#### 図解

```
街灯 ●────30m────●────30m────●────30m────●────30m────●────30m────●────30m────○ 
     0m        30m        60m        90m       120m       150m       180m(街灯なし)

本数: 6本
```

### パターン3: 両端なし（間のみ）の場合

```
計算式: 180m ÷ 30m - 1 = 6 - 1 = 5本
```

#### 図解

```
○────30m────●────30m────●────30m────●────30m────●────30m────●────30m────○ 
0m        30m        60m        90m       120m       150m       180m
(なし)                                                              (なし)

本数: 5本
```

---

## 一般的な解答（最も標準的な解釈）

> **7本**
> 
> 理由：道路の両端に街灯を設置するのが一般的なため

---

## 木の間隔問題との類似

これは数学でよく出る**植木算**の問題と同じです。

| 条件 | 公式 | 答え |
|------|------|------|
| 両端に設置 | 間隔数 + 1 | 6 + 1 = **7本** |
| 片端のみ | 間隔数 | **6本** |
| 両端なし | 間隔数 - 1 | 6 - 1 = **5本** |
| 円形（一周） | 間隔数 | **6本** |

---

## 注意点

### 間隔数の計算

```
180m ÷ 30m = 6

→ これは「間隔の数」= 6個
→ 点（街灯）の数は条件によって変わる
```

### よくある間違い

```
間違い: 180 ÷ 30 = 6本（両端の1本を忘れる）
正しい: 180 ÷ 30 + 1 = 7本（両端に設置）
```

---

## 答え

> **7本（両端に設置する場合）**
> 
> または設置方法の指定により **6本**（片端のみ）または **5本**（両端なし）

標準的な解答: **7本**
