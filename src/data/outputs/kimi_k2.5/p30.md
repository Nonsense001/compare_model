# 知らない場合の対応

## 知らない場合の対応パターン

### 1. 明確に「知らない」と伝える

```
「その情報は持ち合わせていません」
「正確な情報源を確認できないため、回答を控えさせてください」
```

### 2. 関連情報を提供しつつ限界を明記

```
「直接の情報はありませんが、類似のケースでは...」
「これは推測ですが、一般的には...」
```

### 3. 検索・確認を推奨

```
「最新情報は公式サイトをご確認ください」
「専門家への相談をお勧めします」
```

### 4. 推測を推測として提示

```
「確証はありませんが、おそらく...」
「一般的な傾向として...（例外もあります）」
```

---

## 具体的な対応例

| 状況 | 対応 |
|------|------|
| 最新のニュース | 「2024年X月以降の情報は限定的です。最新ニュースサイトをご確認ください」 |
| 個人の非公開情報 | 「その情報はアクセスできません」 |
| 専門的な詳細 | 「専門家の見解を参照することをお勧めします」 |
| 不明確な質問 | 「質問の意図を明確にしていただけますか？」 |

---

## 対応の原則

1. **正直に「知らない」と伝える**
2. **不確かな情報を誤った確信で述べない**
3. **代替情報や検索方法を提示する**
4. **必要に応じて「推測である」と明記する**

---

## 重要な点

> **「知らない」ことを認めることは、回答として適切です。**

誤った情報を自信を持って提供することは、
「知らない」と正直に伝えることよりも
**はるかに有害**です。
