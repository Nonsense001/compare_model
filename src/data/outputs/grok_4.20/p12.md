# 解答：最も背が高いのは誰ですか？

## 与えられた情報
- AはBより背が高い → **A > B**
- BはCより背が低い → **B < C**（つまり **C > B**）

## 分析

2つの条件からわかること：
- AはBより背が高い
- CはBより背が高い

つまり、**AとCは両方ともBより背が高い**ということです。

しかし、**AとCのどちらがより背が高いか**については、一切情報がありません。

## 可能な順位

以下の3パターンがすべて考えられます：

1. **A > C > B**（Aが最も背が高い）
2. **C > A > B**（Cが最も背が高い）
3. **A = C > B**（AとCが同じ身長）

## 結論

**最も背が高いのはAかCのどちらかですが、どちらとも特定できません。**

情報が不足しています。

---

**最終回答**：  
**AとCのどちらか**（正確には判断できません）。

この問題は**不十分な情報**を与えて「結論を急がせる」典型的な論理パズルです。きちんと考えると「わからない」と答えるのが正解です。
