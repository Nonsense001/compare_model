# クラウドとオンプレミスの違い｜中学生向け解説

## 簡単に言うと

| | クラウド | オンプレミス |
|---|---|---|
| **データの場所** | インターネットの向こう側の会社 | 自分たちの会社の中 |
| **イメージ** | レンタルサービス | 自分で持つ家 |

---

## オンプレミス（On-Premises）とは？

**オンプレミス** = 自分たちの「家」の中にコンピュータやサーバーを置いて使う方法。

- 「Premises（プレミセス）」は「建物・敷地」という意味
- 学校のパソコン室のように、**自分たちの場所**に機器を置く
- 買った機械は全部、その場所にずっとある

### 例え話
お家でゲームをするとき、**自分でゲーム機を買って**自宅に置いて遊ぶようなイメージです。

---

## クラウド（Cloud）とは？

**クラウド** = インターネットを通じて、**遠くの会社が持っているコンピュータ**を借りて使う方法。

- 「Cloud（クラウド）」= 雲。インターネットが雲のようにどこかにあるイメージ
- データやサービスは**インターネットの向こう側**にある
- 自分たちは「借りて使う」感じ

### 例え話
Netflixで動画を見るとき、**自分でDVDプレイヤーを持たずに**、ネット越しに動画を借りて見るようなイメージです。

---

## 違いを整理すると

| ポイント | オンプレミス | クラウド |
|---------|-------------|---------|
| **初期費用** | 機器を買うので高い | レンタルなので安く始めやすい |
| **場所** | 自社の部屋や建物 | 使う側には見えない（インターネット上） |
| **メンテナンス** | 自分たちで管理する | サービス提供会社がやってくれることが多い |
| **拡張** | 機材を買い足す必要がある | 契約を変えるだけで増やせる |
| **災害時の備え** | 自社の場所がダメになると困る | 遠くのデータセンターにデータがある場合がある |

---

## 身近な例

### クラウドの例
- **Googleドライブ**：自分のPCに置かずに、ネット上にファイルを保存
- **YouTube**：動画を自分で保存せずに、ネットで見る
- **LINE**：メッセージはLINEのサーバーに保存され、スマホから見ている

### オンプレミスの例
- **学校のパソコン室**：学校の建物の中にPCが置いてある
- **昔の家庭用ゲーム**：カセットやディスクを買って、家のゲーム機で遊ぶ

---

## まとめ

- **オンプレミス**：自分の場所に機械を置いて、自分で管理する
- **クラウド**：インターネット経由で、よその会社のコンピュータを借りて使う

どちらがいいかは、お金、セキュリティ、使い方などによって変わります。今はクラウドを使うサービスが増えていますが、オンプレミスもまだ多くの会社で使われています。
