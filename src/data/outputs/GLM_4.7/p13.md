# 解答

## 問題の整理

5人のうち4人が嘘をついており、犯人は1人です。

## 各人の証言

1. A「Bが犯人だ」
2. B「Cが犯人だ」
3. C「Bは嘘をついている」
4. D「AかBが犯人だ」
5. E「私は犯人ではない」

## 論理的分析

### 仮説：犯人がAである場合

- Aの発言「Bが犯人だ」→ 嘘（正しい：Bは犯人ではない）
- Bの発言「Cが犯人だ」→ 嘘（正しい：Cは犯人ではない）
- Cの発言「Bは嘘をついている」→ 嘘（正しい：Bは真実を言っている）→ しかしBは嘘をついているので矛盾
  - Bは「Cが犯人だ」と言っているが、Aが犯人なのでこれは嘘。したがってCの「Bは嘘をついている」は真実になるはずですが、4人が嘘をついているはずなので矛盾。

**結論：Aは犯人ではない**

### 仮説：犯人がBである場合

- Aの発言「Bが犯人だ」→ 真実
- Bの発言「Cが犯人だ」→ 嘘（正しい：Cは犯人ではない）
- Cの発言「Bは嘘をついている」→ 真実
- Dの発言「AかBが犯人だ」→ 真実（Bが犯人）
- Eの発言「私は犯人ではない」→ 真実

この場合、真実を言っているのはA、C、D、Eの4人で、嘘をついているのはBだけの1人になります。
しかし、問題文では「5人のうち4人が嘘をついている」と言っているので矛盾。

**結論：Bは犯人ではない**

### 仮説：犯人がCである場合

- Aの発言「Bが犯人だ」→ 嘘（正しい：Bは犯人ではない）
- Bの発言「Cが犯人だ」→ 真実
- Cの発言「Bは嘘をついている」→ 嘘（正しい：Bは真実を言っている）
- Dの発言「AかBが犯人だ」→ 嘘（正しい：AもBも犯人ではない）
- Eの発言「私は犯人ではない」→ 真実

この場合、真実を言っているのはBとEの2人で、嘘をついているのはA、C、Dの3人になります。
しかし、問題文では「5人のうち4人が嘘をついている」と言っているので矛盾。

**結論：Cは犯人ではない**

### 仮説：犯人がDである場合

- Aの発言「Bが犯人だ」→ 嘘（正しい：Bは犯人ではない）
- Bの発言「Cが犯人だ」→ 嘘（正しい：Cは犯人ではない）
- Cの発言「Bは嘘をついている」→ 真実（Bは「Cが犯人だ」と嘘をついている）
- Dの発言「AかBが犯人だ」→ 嘘（正しい：AもBも犯人ではない）
- Eの発言「私は犯人ではない」→ 真実

この場合、真実を言っているのはCとEの2人で、嘘をついているのはA、B、Dの3人になります。
しかし、問題文では「5人のうち4人が嘘をついている」と言っているので矛盾。

**結論：Dは犯人ではない**

### 仮説：犯人がEである場合

- Aの発言「Bが犯人だ」→ 嘘（正しい：Bは犯人ではない）
- Bの発言「Cが犯人だ」→ 嘘（正しい：Cは犯人ではない）
- Cの発言「Bは嘘をついている」→ 真実（Bは「Cが犯人だ」と嘘をついている）
- Dの発言「AかBが犯人だ」→ 嘘（正しい：AもBも犯人ではない）
- Eの発言「私は犯人ではない」→ 嘘（正しい：Eは犯人である）

この場合、真実を言っているのはCだけの1人で、嘘をついているのはA、B、D、Eの4人になります。
これは問題文の「5人のうち4人が嘘をついている」という条件と一致します！

## 再確認

犯人がEである場合：
- A：嘘（Bは犯人ではない）
- B：嘘（Cは犯人ではない）
- C：真実（Bは嘘をついている）
- D：嘘（AもBも犯人ではない）
- E：嘘（Eは犯人である）

真実：C（1人）
嘘：A、B、D、E（4人）✓

## 結論

**犯人はEです。**
