# 平成32年5月（2020年5月）の主な出来事

1. **5月25日：緊急事態宣言が全面解除**
   - 安倍晋三首相が東京、神奈川、埼玉、千葉、北海道の5都道県での緊急事態宣言を解除することを発表し、これにより全国での緊急事態宣言が48日ぶりに全面解除された。

2. **5月7日：レムデシビルが国内初の新型コロナ治療薬として特例承認**
   - 厚生労働省が新型コロナウイルス治療薬「レムデシビル」を特例承認した。これが日本国内で初めて承認された新型コロナ治療薬となった。

3. **5月20日：夏の甲子園大会が中止決定**
   - 日本高等学校野球連盟が第102回全国高等学校野球選手権大会（夏の甲子園）の中止を決定した。これは79年ぶり3度目の中止となった。
