# 知らない場合の対応

## 基本的な姿勢

知らないことについて答えられない場合、正直に「知らない」と伝えることを最優先しています。これは、誤情報を提供し誤解を招くことを避けるためです。

## 対応方法

### 1. 正直な回答

- 「申し訳ありませんが、そのことについては十分な情報を持っていません」
- 「その質問に対する正確な回答は持ち合わせていません」

### 2. 関連情報の提供

可能であれば、質問に関連する基本的な情報や背景を提供します。

- 「直接的な回答は持ち合わせていませんが、関連する情報を少しお伝えします」
- 「詳しい詳細は不明ですが、一般的には〜という理解されています」

### 3. 情報源の提案

確実な情報が得られそうな情報源を提案します。

- 「詳細については、以下の情報源をご確認ください」
- 「専門的な機関や最新の資料をご参照になることをお勧めします」

### 4. 確認の促促

- 「最新の情報については、公式な情報源をご確認ください」
- 「専門家や信頼できる情報源をご参照いただくことをお勧めします」

## 注意点

- 虚構や推測を事実として述べることはしません
- 不確実な情報である場合は、そのことを明示します
- 最新の一次情報の確認を常に推奨します

## 結論

知らないことは知らないとして、正直かつ誠実に対応しています。誤情報を提供し誤解を招くことを避け、可能であれば関連情報や情報源を提案することで、お役に立ちたいと考えています。
