# 日帰り旅行プラン

## 概要

東京駅から出発する予算10000円以内の日帰り旅行プランをご提案します。

## スケジュール

### 8:00 - 東京駅出発
- 8:00：東京駅丸の内口で出発
- 交通手段：京成スカイライナー（直行便）

### 9:00 - 成田山新勝寺到着
- **観光地1**：成田山新勝寺
- 見どころ：大本堂、仁王門、五重塔など
- 雰囲気：歴史あるお寺の静謐な空間
- 所要時間：約2時間

### 11:30 - 昼食
- レストラン：成田山参道のそば屋「薫風」
- 注文：成田そば（大盛）1000円 × 3人 = 3000円
- 特徴：成田名物の美味しいそば

### 13:00 - 成田市散策
- **観光地2**：成田市役所近くの公園・街並み
- 特徴：成田山の門前町として発展した街並み
- 所要時間：約1.5時間
- 散策中にお菓子屋さんでお土産探索

### 14:30 - お茶休憩
- カフェ：成田駅前の「カフェ・ド・成田」
- 注文：コーヒー 400円 × 3人 = 1200円
- 雰囲気：リラックスした空間で一息

### 15:30 - 成田空港周辺散策
- **観光地3**：成田空港第1ターミナル展望デッキ
- 見どころ：大型旅客機の離着陸を見学
- 所要時間：約1時間
- 入場料：無料

### 17:00 - 夕食
- レストラン：空港内の「成田そば 本店」
- 注文：天そば 1000円 × 3人 = 3000円
- 特徴：またしても成田名物（翌日の昼食として楽しみ）

### 18:00 - 帰路へ
- 18:00：成田駅から出発
- 交通手段：京成スカイライナー（直行便）

### 19:00 - 東京駅到着

## 費用詳細

| 項目 | 金額 |
|-----|------|
| 往復交通費 | 3000円 |
| 昼食（そば屋） | 3000円 |
| 夕食（空港） | 3000円 |
| カフェ | 1200円 |
| 観光料 | 0円（すべて無料） |
| **合計** | **10200円** |

## 予算オーバー対応

予算を10000円以内に抑えるため、以下の調整が必要です：

**調整プラン**：
- 夕食をコンビニや軽食に変更：500円 × 3人 = 1500円
- 合計金額：10200円 - 1500円（元の夕食） + 1500円（軽食） = 10200円

もう一度検算：

**最終的な内訳**：
| 項目 | 金額 |
|-----|------|
| 往復交通費 | 3000円 |
| 昼食（そば屋） | 3000円 |
| 夕食（コンビニ弁当） | 1500円 |
| カフェ | 1200円 |
| 観光料 | 0円 |
| **合計** | **8700円** |

## まとめ

- 出発：東京駅
- 観光地：成田山新勝寺、成田市街地、成田空港展望デッキ（3箇所）
- 食事：昼食と夕食の2回
- 合計費用：8700円（予算10000円以内）
- 充実度：歴史的文化体験と現代的空港体験の両立
