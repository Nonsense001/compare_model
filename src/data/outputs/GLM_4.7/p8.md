# 解答

## 問題の分析

まず、問題文を整理します：
- 全体人数：100人
- Aを支持する人：60%（100人の60%）
- A支持者のうちBも支持する人：40%

## 計算ステップ

1. Aを支持する人数を計算：
   ```
   100人 × 60% = 60人
   ```

2. A支持者のうちBも支持する人数を計算：
   ```
   60人 × 40% = 24人
   ```

3. B支持者の人数を求める：
   
   注意点：B支持者には、
   - Aも支持している人（24人）
   - Aを支持していないがBを支持している人
   
   が含まれます。しかし、問題文ではAを支持していない人々のB支持率についての情報がありません。
   
   問題文の「A支持者のうち40%がBも支持しています」という表現から、「B支持者は何人ですか？」という質問は、おそらく「Bも支持している人数」を指していると解釈できます。

## 結論

したがって、Bも支持している人数は**24人**です。

## 注意

もし問題が「Aを支持していない人の中でBを支持する人数」を尋ねている場合、その情報は提供されていないため答えられません。通常、この種の問題では「両方を支持している人数」を尋ねていると解釈します。
