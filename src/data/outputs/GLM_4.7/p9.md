# 解答

## 問題の分析

- 全長：180m
- 街灯の設置間隔：30m

## 計算

街灯の設置箇所を考えます：
```
開始点（0m地点）に1本
30m地点に1本
60m地点に1本
90m地点に1本
120m地点に1本
150m地点に1本
180m地点（終了点）に1本
```

街灯の数を計算：
```
街灯の数 = (全長 / 間隔) + 1
         = (180m / 30m) + 1
         = 6 + 1
         = 7本
```

## 重要なポイント

街灯を設置する際、以下の2つの考え方があります：

1. **最初の街灯は起点（0m）に置く**：この場合、終点にも街灯を置くため、間隔の数+1本が必要

2. **最初の街灯は最初の間隔地点（30m）に置く**：この場合、起点には置かないので、単純に「間隔の数」で良い

通常、問題文の「街灯を設置します」という表現からは、道路の開始地点から終了地点まで均等に街灯を配置することが想定されます。そのため、起点と終点の両方に街灯を置くのが一般的です。

## 結論

必要な街灯の数は**7本**です。
