# 解答

## 問題の分析

- 3人でピザを分ける
- Mサイズ：8切れ、1200円
- Lサイズ：12切れ、1800円
- 1人あたり最低4切れ必要

## 考えられる組み合わせ

### 1. Mサイズ1枚（8切れ）
- 合計切れ数：8切れ
- 1人あたりの切れ数：8 ÷ 3 = 2.67切れ
- 結果：4切れ未満なので不適格

### 2. Mサイズ2枚（16切れ）
- 合計切れ数：16切れ
- 1人あたりの切れ数：16 ÷ 3 = 5.33切れ ✓
- 合計金額：1200円 × 2 = 2400円

### 3. Lサイズ1枚（12切れ）
- 合計切れ数：12切れ
- 1人あたりの切れ数：12 ÷ 3 = 4切れ ✓
- 合計金額：1800円 × 1 = 1800円

### 4. Lサイズ2枚（24切れ）
- 合計切れ数：24切れ
- 1人あたりの切れ数：24 ÷ 3 = 8切れ ✓
- 合計金額：1800円 × 2 = 3600円

### 5. Mサイズ1枚 + Lサイズ1枚（20切れ）
- 合計切れ数：8 + 12 = 20切れ
- 1人あたりの切れ数：20 ÷ 3 = 6.67切れ ✓
- 合計金額：1200円 + 1800円 = 3000円

### 6. Mサイズ3枚（24切れ）
- 合計切れ数：24切れ
- 1人あたりの切れ数：24 ÷ 3 = 8切れ ✓
- 合計金額：1200円 × 3 = 3600円

## 結論の比較

| 組み合わせ | 1人あたり切れ数 | 合計金額 |
|-----------|-----------------|---------|
| Mサイズ2枚 | 5.33切れ | 2400円 |
| Lサイズ1枚 | 4切れ | 1800円 |
| Lサイズ2枚 | 8切れ | 3600円 |
| M+L各1枚 | 6.67切れ | 3000円 |
| Mサイズ3枚 | 8切れ | 3600円 |

## 最適解

最も安くなる組み合わせは、**Lサイズ1枚**です。

- 組み合わせ：Lサイズ1枚（12切れ）
- 1人あたりの切れ数：4切れ
- 合計金額：**1800円**

Lサイズ1枚だけで条件（1人あたり4切れ以上）を満たし、最も安価です。
