# あらすじ（宮沢賢治「注文の多い料理店」／400字）

山奥に入った二人の紳士は、猟の途中で道に迷い、飢えのなか「西洋料理店」と看板の出た建物を見つけます。無料のように読める表示に誘われ入店すると、髪を整え、靴を脱ぎ、銃や財布まで預けるなど、奇妙な「注文」が次々と突きつけられます。二人は違和感を覚えつつ従い、香水のような酢の匂いやクリーム塗りを経て、最後には塩を揉み込めと迫られ、ようやく自分たちが料理として選ばれる側だと気づきます。扉の向こうから招かれる恐怖に震えるなか、死んだはずの犬が戻り扉を破り、暗闇の向こうで争いが起き、建物は消え去ります。散らばった持ち物を尻目に、猟師に助けられた二人は団子を食べ、安い山鳥を買って東京へ帰りますが、極度の恐怖で歪んだ顔だけは元に戻らない、という結末です。
