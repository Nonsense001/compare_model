# 日帰り旅行プラン（東京駅出発・目安）

## 前提の置き方

- **予算**：10,000円以内  
- **交通**：往復 **3,000円** 枠（都内＋近郊の電車に収める想定）  
- **食事**：2回  
- **観光**：2箇所以上  

※価格は変動するため、**当日の運賃・入場料で再計算**してください。

## 例：横浜日帰り（イメージ）

| 項目 | 内容 | 目安の費用 |
|------|------|------------|
| 往復交通 | 東京駅〜横浜駅（JR等） | 約1,000円前後×往復で枠内に収めやすい |
| 観光① | みなとみらい周辺（赤レンガ倉庫・プロムナード等、散策中心） | 無料〜 |
| 食事① | ランチ（カレー・定食など） | 約1,200〜1,800円 |
| 観光② | 中華街散策や山下公園 | 無料〜 |
| 食事② | カフェ・軽食または中華街の軽めの食事 | 約800〜1,500円 |
| 予備 | 飲料・土産の小額 | 残り予算内 |

**合計イメージ**：交通3,000＋食事2回で約2,500〜3,500＋α → **1万円以内**に収めやすい構成です。

## 別案（都内だけにする場合）

- **交通**：都内一日乗車券や複数回の地下鉄で **3,000円以内**に設計  
- **観光**：浅草＋スカイツリー周辺、または上野公園＋博物館（有料は1つに抑える）  
- **食事**：立ち食いそば＋カジュアルな夕食 など

## メモ

「交通費3000円」が**厳密に上限**なら、行き先を**京浜東北・山手線圏**に寄せると安全です。予約が要る施設は事前確認を。
