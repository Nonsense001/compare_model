# 傘の売上急増：考えられる理由（仮説）と思考過程

## 思考の進め方

まず「**売上＝需要×単価×販売量**」に分解し、次に**急増**なので「短期の変化要因」を探します。最後に、**仮説ごとに反証する観測**（天気・競合・販促・調査バイアス）を考えます。

## 仮説の例

1. **気象要因（最も素直）**  
   - 雨・台風の日数が増えた、または予報精度が上がり「傘を持つ」行動が増えた。  
   - **反証チェック**：過去同期と比較して降水量・雨天日数が本当に増えたか。

2. **イベント・人流**  
   - 観光客やイベント来場者が増え、**駅・商業施設の即購買**が増えた。  
   - **反証チェック**：来街者数、イベントの有無、店舗立地の人流データ。

3. **販売チャネル・商品**  
   - コンビニ・ECの**割引・セット販売**、**軽量・折りたたみ**の新商品がヒットした。  
   - **反証チェック**：単価・点数・SKU別の販売数量。

4. **統計・データの取り方**  
   - 「急増」が**前年比の偶然**、**サンプル店舗の偏り**、**在庫切れからの復帰**などの**見方の問題**かもしれない。  
   - **反証チェック**：定義（同店舗比較か）、季節調整、欠落データ。

5. **代替品の不調**  
   - レインコート・防水ウェアの供給遅延、価格高騰で、**傘に代替**した。  
   - **反証チェック**：関連カテゴリの売上・在庫。

## まとめ

単一の「正解」より、**複数仮説を並べ、観測可能なデータで順に潰す**のが現実的です。天候要因が強い場合は短期、商品・チャネル要因なら中期的な施策が続く、など**持続性**の見立ても分けます。
