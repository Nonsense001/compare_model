# 平成32年5月に起きた出来事について

**結論：「平成32年5月」という日付は、日本の元号の扱い上、存在しません。**

## 理由

- **平成は31年で終了**しています（2019年4月30日が平成最後の日）。
- その翌日から**令和**が始まります（2019年5月1日〜）。

したがって、**「平成32年5月」に起きた出来事を列挙する**という前提そのものが成立しません。

## もし意図が「2020年5月」など別の日付なら

カレンダー上の **2020年5月** に起きた出来事としては、例として次のようなものが挙げられます（例示であり網羅ではありません）。

1. **新型コロナウイルス感染症**に関する各国の行動制限・宣言の継続・拡大  
2. **日本国内**でも緊急事態宣言の対象・期間などが議論・運用された時期  
3. その他、当該月のニュースとして報じられた政治・経済・社会の出来事

ご希望が「令和2年5月」や西暦での指定であれば、その前提に合わせて整理し直せます。
